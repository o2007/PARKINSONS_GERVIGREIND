# Parkinsons_gervigreind_spa

Lokaverkefni í RAF620M – Inngangur að vélrænu námi og gervigreind.

## Lýsing
Þetta verkefni beitir gervigreindar- og vélrænum námsaðferðum til að greina og spá fyrir um Parkinson's sjúkdóm út frá gögnum.

## Uppbygging verkefnis
```
Parkinsons_gervigreind_spa/
│
├── data/           # Gagnasöfn
├── notebooks/      # Jupyter notebooks
├── src/            # Python kóði
└── report/         # Skýrslur
```

## Skref verkefnis
1. Inngangur
2. Gagnagrunnur
3. Pre-processing
4. Niðurstöður úr einföldu módeli
5. Næstu skref
6. Lokaorð
